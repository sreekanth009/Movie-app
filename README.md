# Movie-app
Movies information and related movies list
 
 Steps for local setup
 
## Local Setup
 1. Local
    clone the repo
    cd project (Movie-app)
    npm install
    npm run dev 
    now your local will be up and running.


## Frontend Commands

   1. Local

          cd Frontend
          npm install
          npm start
          npm run local

   2. Dev

          cd Frontend
          npm install
          npm run build:dev

   3. Production

          cd Frontend
          npm install
          npm run build:prod
